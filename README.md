# Piano Virtual (Mapeamento de Eventos e Interface)

Este projeto é a implementação de um piano virtual simples usando **JavaScript, HTML e CSS**, com foco na manipulação de **eventos do usuário** e no mapeamento de interface/dados.

## Objetivo Técnico

O principal objetivo foi desenvolver uma aplicação que traduz a **interação física do usuário (clique ou tecla pressionada)** em uma **resposta digital (som)**, simulando um ambiente de **entrada de dados e feedback instantâneo**.

Os pontos técnicos focados foram:

1.  **Manipulação de Eventos:** Uso de *Event Listeners* (cliques e eventos de teclado) para detectar a entrada do usuário de forma eficiente.
2.  **Mapeamento de Dados:** Criação de uma lógica para mapear cada tecla da interface a um **arquivo de áudio específico**, garantindo a correspondência exata entre input e output.
3.  **Controle de Fluxo (JavaScript):** Gerenciamento do ciclo de vida dos sons (início e parada) e controle de *timing*.
4.  **Integração Multimídia:** Integração de arquivos de áudio ao código para criar uma experiência interativa.

## Funcionalidades

* Piano virtual com teclas clicáveis.
* Reprodução de som correspondente ao clique na tecla.

## Tecnologias utilizadas

* **JavaScript (Vanilla):** Lógica principal, Manipulação de Eventos e Controle de Áudio.
* **HTML5:** Estrutura e marcação de elementos de interface.
* **CSS3:** Estilização das teclas e da interface.
